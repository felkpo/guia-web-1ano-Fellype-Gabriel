Fundamentos da Web

HTML (HyperText Markup Language)

HTML é a linguagem de marcação que define a estrutura do conteúdo web. É como o esqueleto de uma página, determinando:

  • Hierarquia do conteúdo
  • Organização dos elementos
  • Semântica da informação

CSS (Cascading Style Sheets)

CSS é responsável pela estilização visual do site, controlando:

  • Cores e tipografia
  • Layout e posicionamento
  • Animações e transições
  • Responsividade

JavaScript

JavaScript traz interatividade e dinamismo às páginas web, permitindo:

  • Manipulação do DOM
  • Requisições assíncronas
  • Validações e cálculos
  • Criação de interfaces dinâmicas

Boas Práticas Profissionais

Acessibilidade

  • Use atributos ARIA quando necessário
  • Forneça textos alternativos para imagens
  • Garanta navegação por teclado
  • Mantenha bom contraste de cores

Semântica

  • Use tags HTML5 semânticas (header, nav, main, section, etc.)
  • Estruture corretamente os níveis de títulos
  • Nomeie classes e IDs de forma significativa

Responsividade

  • Design mobile-first
  • Use media queries estrategicamente
  • Imagens e vídeos flexíveis
  • Grid e Flexbox para layouts adaptáveis

Organização e Versionamento

  • Estrutura clara de arquivos e pastas
  • Uso de Git para controle de versão
  • Documentação do projeto
  • Padronização de código (ESLint, Prettier)

Panorama de Tecnologias Web

Front-end

  • Frameworks: React, Vue.js, Angular
  • Pré-processadores CSS: Sass, Less
  • Bundlers: Webpack, Vite

Back-end

  • Node.js e Express
  • Python e Django/Flask
  • PHP e Laravel
  • Java e Spring

Bancos de Dados

  • Relacionais: MySQL, PostgreSQL
  • NoSQL: MongoDB, Redis
  • Serviços: Firebase, Supabase

DevOps

  • CI/CD: Jenkins, GitHub Actions
  • Containers: Docker, Kubernetes
  • Cloud: AWS, Azure, GCP

Testes

  • Unitários: Jest, Mocha
  • E2E: Cypress, Selenium
  • Integração: Supertest
